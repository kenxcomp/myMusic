根据用户音乐偏好 + 指定情绪/曲风，推荐 20 首歌曲，覆盖「推荐歌单」。

用户输入参数: $ARGUMENTS

**$ARGUMENTS 为情绪或曲风关键词**，例如：快乐、伤感、治愈、燃、放松、怀旧、浪漫、电子、摇滚、民谣等。

## 执行流程

### 第一步：校验登录状态

```bash
ncm-cli login --check
```
如果未登录，执行 `ncm-cli login --background` 并将链接给用户，等用户确认登录后再继续。

### 第二步：查询用户音乐偏好

并行执行以下命令，获取用户最新的音乐偏好数据：

1. `ncm-cli user history --limit 20 --userInput "查看最近播放记录了解偏好"` — 最近播放记录
2. `ncm-cli user favorite --userInput "查看红心歌单信息"` — 获取红心歌单信息（记录红心歌单的加密 ID，后续心动推荐需要）

提取并分析用户最近听歌的风格标签倾向，作为推荐参考。

### 第三步：设计情绪搜索策略

根据 `$ARGUMENTS` 指定的情绪/曲风，结合用户偏好，设计 **3-5 个搜索关键词**。

**关键词设计原则：**
- 将情绪映射为具体的音乐风格、代表性歌手、经典歌曲名
- 结合用户偏好中的语种倾向（华语/英文/日韩等）
- 每个关键词尽量覆盖不同维度，避免重复

**情绪映射示例（仅供参考，应结合用户偏好灵活调整）：**
- 快乐 → 欢快、阳光、活力、甜蜜、派对
- 伤感 → 悲伤、失恋、思念、离别、雨天
- 治愈 → 温暖、舒缓、轻音乐、陪伴、晚安
- 燃 → 热血、激情、运动、摇滚、电音
- 放松 → 慵懒、午后、咖啡、bossa nova、轻爵士
- 怀旧 → 经典、老歌、回忆、青春、校园
- 浪漫 → 甜蜜、情歌、约会、恋爱、告白

### 第四步：搜索情绪匹配歌曲

用设计好的关键词逐一搜索：

```bash
ncm-cli search song --keyword "<关键词>" --userInput "搜索<情绪>风格的歌曲"
```

从每组搜索结果中筛选 **visible 为 true** 的歌曲，收集候选列表。

### 第五步：心动推荐补充

从第四步的候选歌曲中挑选一首最符合目标情绪的歌曲作为基准，执行心动推荐：

```bash
ncm-cli recommend heartbeat --playlistId <红心歌单加密ID> --songId <基准歌曲加密ID> --count 50 --userInput "基于<情绪>风格歌曲进行心动推荐"
```

从心动推荐结果中筛选与目标情绪匹配的歌曲，加入候选列表。

### 第六步：最终筛选 20 首

从所有候选歌曲中筛选最终 20 首：

1. **visible 必须为 true**
2. 排除用户已红心的歌曲（liked 为 true 的跳过，确保推荐新鲜感）
3. 去重（同一首歌可能在多个搜索结果中出现）
4. 优先选择与 `$ARGUMENTS` 情绪最匹配的歌曲
5. 兼顾多样性：不同歌手、不同子风格，避免同一歌手超过 2 首

如果不足 20 首，可追加关键词搜索或换基准歌曲再次心动推荐补足。

### 第七步：查找「推荐歌单」

```bash
ncm-cli playlist created --userInput "查找推荐歌单"
```

从返回结果中找到名为「推荐歌单」的歌单，记录其加密 ID。

如果「推荐歌单」不存在，则创建：
```bash
ncm-cli playlist create --playlistName "推荐歌单" --userInput "创建推荐歌单"
```

### 第八步：清空旧歌曲

获取「推荐歌单」当前的所有歌曲：
```bash
ncm-cli playlist tracks --playlistId <推荐歌单加密ID> --limit 500 --userInput "获取推荐歌单当前歌曲"
```

如果歌单内有歌曲，批量删除：
```bash
ncm-cli playlist remove --playlistId <推荐歌单加密ID> --songIdList '["id1","id2",...]' --userInput "清空推荐歌单"
```

### 第九步：添加新推荐歌曲

将筛选出的 20 首歌曲加密 ID 批量添加：
```bash
ncm-cli playlist add --playlistId <推荐歌单加密ID> --songIdList '["id1","id2",...]' --userInput "添加<情绪>风格推荐歌曲到推荐歌单"
```

### 第十步：输出结果

以表格形式展示推荐结果，包含：
- 序号
- 歌曲名（带网易云链接，使用明文 ID：`https://music.163.com/#/song?id=<明文ID>`）
- 艺人
- 情绪/风格标签

最后附上：
- 本次情绪主题：`$ARGUMENTS`
- 歌单链接：`https://music.163.com/#/playlist?id=<推荐歌单明文ID>`
